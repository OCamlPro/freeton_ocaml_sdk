Library freeton-ocaml-sdk

This project implements an OCaml binding to the TON-SDK and a set of tools in OCaml to interact with Free TON, in particular the 'ft' command-line wallet.

This library exposes the following toplevel modules:

  • Graphql
  • Ton_abi
  • Ton_action
  • Ton_block
  • Ton_client
  • Ton_crypto
  • Ton_encoding
  • Ton_request
  • Ton_rpc
  • Ton_sdk
  • Ton_types
